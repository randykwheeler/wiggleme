**Wiggle Me!** is a high-performance, system-native Windows utility designed to keep your screen active with ultra-realistic, human-like mouse movements.

## ‚ú® Features

- **Human Mode**: Uses advanced Cubic Bezier curves to mimic the natural arc and micro-variations of a human hand.
- **Chaos Mode!**: A high-velocity "crazy" mode for when you need total screen dominance.
- **Manual Override Safety**: Instantly stops wiggling the moment you move your physical mouse. No more getting "locked out" by Chaos Mode.
- **Animated Mascot**: A funny, reactive mascot that wiggles along with your mouse.
- **Lightning Fast**: Built with pure Win32 API and C++ for zero overhead and maximum efficiency.
- **Native Look**: Follows your Windows system theme and colors.
